# print-arxiv

This is a python script that crawls arXiv search results and compiles useful information into one PDF file.  

Reads in:  
`http://arxiv.org/find/*`

Sample output:  
![](http://i.imgur.com/Pk3WK6p.png)

Requires:  
* python
* [python-requests](http://docs.python-requests.org/en/latest/)
* pandoc

Tested on Windows.
